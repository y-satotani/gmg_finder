unsigned int  degree

次数dの平衡木における頂点vの深さを求める．

次の式で計算される．

\begin{align*} & 0 &\:& v = 0 \\ & \mbox{log}_{d-1}{\left(\frac{(v-1)(d-2)}{d}+1\right)} + 1 &\:& v > 0 \end{align*}

unsigned int gmgf::n_vert_btree ( unsigned int  depth,
unsigned int  degree 
)

深さと次数から平衡木の頂点数を求める．

次の式で計算される．

\begin{align*} & 1 &\:& \mbox{dep} = 0 \\ & \mbox{deg}+1 &\:& \mbox{dep} = 1 \\ & \frac{\mbox{deg}\left((\mbox{deg}-1)^{\mbox{dep}}-1\right)} {\mbox{deg}-2}+1 &\:& \mbox{dep} > 1 \end{align*}

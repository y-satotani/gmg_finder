Detailed Description

あるグラフ初期化子の候補辺リストをソートして返すグラフ初期化子．

あるグラフ初期化子に対して，その候補辺リストを次の規則で並べ替える．

  1. 各頂点 に対して，候補辺リストにおける出現回数を求める．
  2. 出現回数がもっとも小さい頂点と接続する辺を取り出し， 新たな候補辺リストに加える．
  3. 元の候補辺リストが空になるまで繰り返す．

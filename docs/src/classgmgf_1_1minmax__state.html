Detailed Description

最大グラフで直径の下界を導出し，枝刈りを実現した状態．

最大グラフとは，未判定の辺すべてを探索中のグラフに追加した グラフである．例えば，

の最大グラフは，破線で示した辺をすべて追加して，

である．このとき，元になったグラフを最小グラフという．

このグラフの直径が，直径の下界である． 直径の下界が条件を満たさなければ枝刈りを行う．
